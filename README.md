# Detection targets with radar — Визуализация и детекция целей с помощью данных радара

Приложение для загрузки, визуализации и детекции целей по данным радара. Построено на Qt и QCustomPlot.

## Возможности

- **Загрузка бинарных файлов** радарных данных (формат .bin)
- **График амплитуды** в зависимости от угла азимута
- **Детекция объектов** по локальным максимумам, превышающим порог
- **Настраиваемый порог** (0–1) для фильтрации ложных срабатываний
- **Таблица обнаруженных целей** с углом и амплитудой
- **Интерактивный график** — масштабирование и перемещение (zoom/pan)

## Требования

- [Qt](https://www.qt.io/) (core, gui, widgets, printsupport)
- Компилятор с поддержкой C++11
- [QCustomPlot](https://www.qcustomplot.com/) — включён в проект

## Сборка

```bash
qmake RadarTest.pro
make
```

Либо откройте `RadarTest.pro` в Qt Creator и соберите проект.

## Использование

1. Запустите приложение.
2. Нажмите **«Выбрать файл»** и укажите бинарный файл радарных данных (*.bin).
3. На графике отобразятся:
   - синяя линия — амплитуда сигнала;
   - красная пунктирная линия — порог детекции;
   - зелёные точки — обнаруженные объекты (локальные максимумы выше порога).
4. Меняйте порог в спинбоксе (0–1), чтобы регулировать чувствительность детекции.
5. Обнаруженные цели выводятся в таблицу справа (угол и амплитуда).

## Формат файла данных

Бинарный файл с заголовком:

| Поле | Размер | Описание |
|------|--------|----------|
| `head` | 4 байта | Должно быть `"HEAD"` |
| `start_angle_int` | 2 байта | Начальный угол (целая часть) |
| `start_angle_frac` | 2 байта | Начальный угол (дробная часть) |
| `end_angle_int` | 2 байта | Конечный угол (целая часть) |
| `end_angle_frac` | 2 байта | Конечный угол (дробная часть) |
| `step_angle_int` | 2 байта | Шаг угла (целая часть) |
| `step_angle_frac` | 2 байта | Шаг угла (дробная часть) |
| `data_id` | 4 байта | Должно быть `"DATA"` |

За заголовком идут `int16_t` амплитуды (нормализуются делением на 32768).
